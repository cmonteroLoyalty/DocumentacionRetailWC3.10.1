Selector name

This widget has the following selector name: bb-actions-product-notifications-settings-widget

Required imports

The following imports are required to set up and customize this widget:

  • Widget module import

    Import the following modules into the module of your wrapper widget:

    import { ActionsProductNotificationsSettingsWidgetModule } from "@backbase/actions-product-notifications-settings-widget-ang";

Extension slots

This widget has the following extension slots:

  1. bbActionsProductItemCardCustomizableDirective

  2. bbActionsProductSettingsPageCustomizable

  3. bbActionsProductSettingsPageErrorNotificationHeaderCustomizable

  4. bbActionsProductSettingsPageErrorNotificationMessageCustomizable

  5. bbActionsAccountBalanceRecipeCustomizableDirective

  6. bbActionsTransactionsRecipeCustomizableDirective

  7. bbActionsNotificationChannelsCustomizableDirective

bbActionsProductItemCardCustomizableDirective

This extension slot includes the following default template:

<ng-template
  bbActionsProductItemCardCustomizableDirective
  let-hostRef
  let-account="context"
>
  <div>
    <div data-role="card-title" class="bb-text-bold">
      {{ hostRef.getAlias(account) }}
    </div>
    <div data-role="card-sub-title" class="text-small bb-text-support bb-block bb-block--xl">
      {{ hostRef.retrieveAccountNumber(account) | bbIban | paymentCardNumber: productNumberFormat }}
    </div>
  </div>
</ng-template>

Implementation example

The following examples show the implementation of this widget:

  • Wrapper widget module

    actions-product-notifications-settings-extended.module.ts
    import { NgModule } from '@angular/core';
import { CommonModule } from '@angular/common';
import { BackbaseCoreModule } from '@backbase/foundation-ang/core';
import { ActionsProductNotificationsSettingsWidgetModule } from '@backbase/actions-product-notifications-settings-widget-ang';
import { ActionsProductNotificationsSettingsExtendedComponent } from './actions-product-notifications-settings-extended.component';

@NgModule({
  declarations: [ActionsProductNotificationsSettingsExtendedComponent],
  imports: [
    CommonModule,
    BackbaseCoreModule.withConfig({
      classMap: { ActionsProductNotificationsSettingsExtendedComponent },
    }),
    ActionsProductNotificationsSettingsWidgetModule,
  ],
})
export class ActionsProductNotificationsSettingsExtendedModule {}

  • Wrapper widget component

    actions-product-notifications-settings-extended.component.ts
    import { Component, Output, Input, EventEmitter } from '@angular/core';

@Component({
  selector: 'bb-actions-product-notifications-settings-extended',
  templateUrl: './actions-product-notifications-settings-extended.component.html',
})
export class ActionsProductNotificationsSettingsExtendedComponent {
  @Input() selectedAccountId: string | undefined;

  /**
   * @deprecated since 3.6.2 - will be removed in 3.8.0
   * design without save/cancel doesn't implement navigation
   * Event emitted when this product notifications settings saved successfully
   */
  @Output() productNotificationsSettingsSaved = new EventEmitter<void>();

  /**
   * @deprecated since 3.6.2 - will be removed in 3.8.0
   * design without save/cancel doesn't implement navigation
   * Event emitted on editing cancellation
   */
  @Output() productNotificationsSettingsCancelled = new EventEmitter<void>();

  onProductNotificationsSettingsSave() {
    this.productNotificationsSettingsSaved.emit();
  }

  onProductNotificationsSettingsCancel() {
    this.productNotificationsSettingsCancelled.emit();
  }
}

</bb-actions-product-notifications-settings-widget>
