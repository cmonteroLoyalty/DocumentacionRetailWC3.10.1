Native mobile widgets

The Android and iOS widgets included in Transactions are:

  • View transactions, turnovers, and spending/income analysis

    This package includes:

    • View transaction details

    • Filter transactions

    • Filter transactions by amount range

    • Filter transactions by date range

    • Filter transactions by transaction type

    • View check images

View transactions, turnovers, and spending/income analysis

This widget displays a list of transactions.

The list is ordered by date and shows the amount, beneficiary, recipient, and currency. It also allows a user to search for transactions where the query will be sent to the server. When tapping on one of the transactions, a pub/sub event can be fired which can, for example, trigger a transactions details widget to be displayed. A product summary can be shown at the top of the list so as to indicate for which product the transactions are being shown.

This widget uses the Transactions client.

Features

  • Retrieves a list of transactions with pagination.

  • Retrieves a list of transactions with category name.

  • Displays the details of a transaction.

  • Retrieves the product summary of the selected product.

  • Searches transactions by the given query and parameters for pagination. The implementation expects a search bar to be implemented on the app level following theÂ search bar framework. The search query will be sent and handled by the service.

  • Retrieves turnovers (incoming/outgoing transactions) for a given set of products.

  • Retrieves category totals.

  • Enables you to list pending transactions either on top of the list or mixed within the history (billed) transactions.

  • Enables you to list transaction address and show the geolocation in a map.

  • Enables you to add, edit, and delete notes for a transaction.

  • Enables you to view the running balance when a transaction is recorded.

  • Enables you to update the view with the filter information.

  • Enables you to send the filter information to the Filter transactions widget.

  • Enables you to view the merchant name and icon of a transaction.

  • Enables you to filter transactions by the given filter criteria.

View transaction details

In the View transaction details widget, the bank customer views the details of a single transaction.

This widget is designed to be integrated with the View transactions widget.

Features

This widget provides the bank customer with the following features:

  • Access detailed information on a single transaction, such as type, description, etc.

  • View notes added to a transaction previously.

  • View images of both the front and the back of cleared checks for check withdrawals.

Filter transactions

In the Filter transactions widget, the bank customer filters their transactions list based on incoming and outgoing transactions. In order to provide additional filtering options to the bank customer, this widget supports integration with the following widgets:

  • Filter transactions by transaction type

  • Filter transactions by amount range

  • Filter transactions by date range

The results received from this widget are meant to be used in combination with the View transactions, turnovers and spending/income analysis widget. The Filter transactions widget allows users to select the filter options and the View transactions widget to search by the given filter criteria.

Features

THis widget has the following features:

  • Ability to filter transactions by incoming and outgoing transactions.

  • Ability to update the view with the filter information, which is received from the View transactions widget.

  • Ability to update the view with amount range information, which is received from the Filter transactions by amount range widget.

  • Ability to update the view with date range information, which is received from the Filter transactions by date range widget.

  • Ability to update the view with transaction type information, which is received from Filter transactions by transaction type widget.

  • Ability to reset filters.

  • Ability to apply the filters in the View transactions widget.

  • API - This widget is part of the TransactionsWidget package. For package API reference, see Android / iOS

Filter transactions by amount range

In the Filter transactions by amount range widget, the bank customer specifies a range of transaction amount.

To specify the amount range, the bank customer enters numbers into the From and To fields, and taps Done. The bank customer then gets redirected to the Filter transactions widget, where the specified transaction amount filter is visible.

This widget is designed to be integrated with the Filter transactions widget.

Features

This widget provides the bank customer with the following features:

  • Specify an amount range to view transactions within that range.

  • Reset the filter.

Filter transactions by date range

In the Filter transactions by date range widget, the bank customer specifies a range of booking dates.

To specify the date range, the bank customer enters dates into the From and To fields, and taps Done. The bank customer then gets redirected to the Filter transactions widget, where the specified transaction date filter is visible.

This widget is designed to be integrated with the Filter transactions widget.

Features

This widget provides the bank customer with the following features:

  • Specify an date range to view transactions within that booking date range.

  • Reset the filter.

Filter transactions by transaction type

In the Filter transactions by transaction type widget, the bank customer filters their transactions in the list by transaction type. Transaction types are for example: ATM, ACH, bill payment, cash, etc.

Once the bank customer selected a transaction type, it is highlighted. To apply the selected filter, the bank customer taps Done. The bank customer then gets redirected to the Filter transactions widget, where the applied transaction type filter is visible.

This widget is designed to be integrated with the Filter transactions widget.

Features

This widget provides the bank customer with the following features:

  • Select a single transaction type to view the transactions for that type.

  • Deselect a previously selected transaction type.
