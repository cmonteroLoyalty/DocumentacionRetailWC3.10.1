View bill payee summary

This page includes the following information:

  • Selector name

  • Required imports

  • Extension slots

  • API

  • Implementation example

  • Preferences

Selector name

This widget has the following selector name: <bb-billpay-payee-summary-widget>

Required imports

The following imports are required to set up and customize this widget:

  • Widget module import

    import { BillpayPayeeSummaryWidgetModule } from '@backbase/billpay-payee-summary-widget-ang';
    NOTE

    All Bill Pay widgets are closely related to each other, so it is recommended to use them together.

Extension slots

This widget has the following extension slots:

  1. bbBillpayPayeeSummaryWidgetCustomizable

  2. bbBillpayPayeeStepperCustomizable

  3. bbBillpayPayeeInformationCustomizable

  4. bbBillpayPayeeSummaryWidgetNotificationHeaderCustomizable

  5. bbBillpayPayeeSummaryWidgetNotificationMessageCustomizable

API

The following table shows the public class members of the host component that you access from each extension slot:

Component Extension slot identifier Public class member Description

BillpayPayeeSummaryWidgetComponent

bbBillpayPayeeSummaryWidgetCustomizable

nextPayee()

Increments payeeIndex: number. Updates payeeSummary: PayeeSummary with the indexed payee. Retrieves and updates fullManualPayeeSummary: Payee if the newly selected payee is manual.

previousPayee()

Decrements payeeIndex: number. Updates payeeSummary: PayeeSummary with the indexed payee. Retrieves and updates fullManualPayeeSummary: Payee if the newly selected payee is manual.

handleEditPayee()

If the payee is electronic, emits the event editElectronicPayee: EventEmitter<string> with the payee ID. Output property: output.editElectronicPayee. Otherwise emits the event editManualPayee: EventEmitter<string> with the payee ID. Output property: output.editManualPayee.

handleDeletePayee()

Deletes the selected payee if there are no scheduled payments.

handleMakePayment()

Emits the event payPayeeOneOff: EventEmitter<PayeeSummary>. Output property: output.payPayeeOneOff.

addPayee()

Emits the event managePayee: EventEmitter<PayeeSummary>. Output property: output.managePayee.

BillpayPayeeInformationComponent

bbBillpayPayeeInformationCustomizable

handleEditPayee()

Emits the event editPayee: EventEmitter. Requests the widget to navigate to the manage payee widget.

handleDeletePayee()

Emits the event deletePayee: EventEmitter. Requests the widget to delete the selected payee.

BillpayPayeeStepperComponent

bbBillpayPayeeStepperCustomizable

handleGetPreviousPayee()

Emits the event previousPayee: EventEmitter. Requests the widget to navigate to the previous payee.

handleGetNextPayee()

Emits the event nextPayee: EventEmitter. Requests the widget to navigate to the next payee.
