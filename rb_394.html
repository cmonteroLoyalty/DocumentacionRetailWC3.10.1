Retail Banking Widget Collection v3.9.4

The highlights for this release are:

  • Capabilities

    • Payments v3.9.4

      • Transaction signing is now available.

    • Self Enrollment v3.9.4

      • Bug fix

    • User Self Service v3.9.4

      • Bug fix

  • Migration: No migration steps required for this release.

To find out which Backbase product versions have been verified together, see Backbase product compatibility.

Payments v3.9.4

This release note explains changes to this capability.

New features

This release provides the following new features:

  • Web: Transaction signing

    Transaction signing is now supported out of the box in the Make a transfer form.

Bug fixes

This release includes the following bug fixes:

  • MAINT-11619: There is no user feedback when account number for predefined payee is incorrect

    There was no feedback on the UI when the backend returned an error. An error message is now added to the UI.

  • MAINT-11607: There is no user feedback while executing rush payment results is unsuccessful

    There was no feedback on the UI when the backend returned an error. An error message is now added to the UI.

  • MAINT-11696: Connecting to external accounts misplaced decimal

    The entered values are now sent to the endpoint with correct format.

  • MAINT-11781: For Vietnamese locale, the amount group separator is considered as a decimal separator for the bb-currency-input-ui input value

    The entered values are now set with the correct format.

Self Enrollment v3.9.4

This release note explains changes to this capability.

Bug fixes

This release includes the following bug fixes:

  • MAINT-10207: Localization keys should not have return characters

    To prevent issues with localization tools, the widgets in the Self Enrollment capability have been updated to ensure that no localization keys have return characters.

User Self Service v3.9.4

This release note explains changes to this capability.

Bug fixes

This release includes the following bug fixes:

  • MAINT-11743: Add extension point for error messages in the Manage user security widget.

    Previously, you couldn’t change the error message the widget displayed to explain why a user couldn’t change their password. The widget always displayed the notification "Could not update password. Incorrect login credentials or password policy not met."

    To enable you to change this notification, it has been moved out to the parent container, which allows extension.
