View transactions list

This page includes the following information:

  • Selector name

  • Required imports

  • Extension slots

  • Implementation example

  • Preferences

Selector name

This widget has the following selector name: <bb-transactions-list-widget>.

Required imports

The following imports are required to set up and customize this widget:

  • Widget module import

    Import the following modules into the module of your wrapper widget:

    import { TransactionsListWidgetModule } from '@backbase/retail-ang/transactions';

  • Widget component import

    Import the following components into the module of your wrapper widget:

    import { TransactionsListWidgetComponent } from '@backbase/retail-ang/transactions';

  • Data module imports

    Import the following data modules into the module of your wrapper widget:

    import { AccountDataHttpModule } from '@backbase/data-ang/account';
import { TransactionsDataHttpModule } from '@backbase/data-ang/transactions';
import { CategoriesManagementDataHttpModule } from '@backbase/data-ang/categories-management';
