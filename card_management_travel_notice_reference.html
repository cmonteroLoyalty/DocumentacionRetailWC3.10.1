Manage travel notice

This page includes the following information:

  • Selector name

  • Required imports

  • Extension slots

  • Implementation example

  • Preferences

Selector name

This widget has the following selector name: <bb-cards-travel-notice-widget>.

Required imports

The following imports are required to set up and customize this widget:

  • Widget module import

    Import the following modules into the module of your wrapper widget:

    import { CardsTravelNoticeModule } from '@backbase/retail-ang/cards';

  • Widget component import

    Import the following components into the module of your wrapper widget:

    import { CardsTravelNoticeWidgetComponent } from '@backbase/retail-ang/cards';

  • Data module imports

    Import the following data modules into the module of your wrapper widget:

    import { CardsDataHttpModule } from '@backbase/data-ang/cards';

This widget has the following extension slots:

  1. bbTravelNoticeEmptyStateContainerCustomizable
