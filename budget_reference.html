Manage budgets

This page includes the following information:

  • Selector name

  • Required imports

  • Extension slots

  • Implementation example

  • Preferences

Selector name

This widget has the following selector name: <bb-budget-widget>.

Required imports

The following imports are required to set up and customize this widget:

  • Widget module import

    Import the following modules into the module of your wrapper widget:

    import { BudgetWidgetModule } from '@backbase/retail-ang/pfm';

  • Widget component import

    Import the following components into the module of your wrapper widget:

    import { BudgetWidgetComponent } from '@backbase/retail-ang/pfm';

  • Data module imports

    Import the following data modules into the module of your wrapper widget:

    import { BudgetingDataHttpModule } from '@backbase/data-ang/budgeting';
import { CategoriesManagementDataHttpModule } from '@backbase/data-ang/categories-management';

Extension slots

This widget has the following extension slots:

  1. bbBudgetsHeaderCustomizable

  2. bbBudgetCardCustomizable

  3. bbBudgetCreateModalCustomizable

  4. bbBudgetUpdateModalCustomizable

  5. bbBudgetConfirmationModalCustomizable

  6. bbBudgetNotificationCustomizable

bbBudgetsHeaderCustomizable

This extension slot includes the following default template:

<ng-template bbBudgetsHeaderCustomizable let-hostRef>
  <bb-budgets-header-ui
    (createBudget)="hostRef.handleCreate()"
  ></bb-budgets-header-ui>
</ng-template>

bbBudgetCardCustomizable

This extension slot includes the following default template:

<ng-template bbBudgetCardCustomizable let-hostRef let-budget="context">
  <bb-budget-card-ui
    class="bb-block bb-block--xl col-md-6 col-lg-4"

</ng-template>

Implementation example

  • Wrapper widget module

    NewExtensionSlot.module.ts
    import { NgModule } from '@angular/core';
import { CommonModule } from '@angular/common';
import { BudgetWidgetModule } from '@backbase/retail-ang/pfm';
import { BudgetsMySelectorComponent } from './NewExtensionSlot.component'

@NgModule({
  declarations: [BudgetsMySelectorComponent],
  imports: [
    CommonModule,
    BudgetWidgetModule,
  ],
})
export class CustomAngModule {}

  • Wrapper widget component

    NewExtensionSlot.component.ts
    import { Component } from '@angular/core';
import { CopyRoutes } from '@backbase/core-ang';
import { BudgetWidgetComponent } from '@backbase/retail-ang/pfm';

@Component({
  selector: 'bb-my-selector',
  templateUrl: 'my-selector.component.html',
})
@CopyRoutes(BudgetWidgetComponent)
export class BudgetsMySelectorComponent {
  // Include your customization here
}

    BudgetWidgetComponent is the name of the Manage Budgets widget component.

  • Custom template

    NewExtensionSlot.component.html
    <bb-budget-widget></bb-budget-widget>

<!-- Override the empty state for the Manage Budgets Widget -->
<ng-template bbExtensionSlot let-hostRef>
  <!-- Place your customization here -->
</ng-template>

Preferences

The widget has the following preferences that managers or administrators configure in Experience Manager:

Title and description Name Admin[a] Default value

Show budget values as percentages

It shows budget values as percentages.

showPercentage

x

false

Limit for the safe (green) zone

It sets a limit for budgets in percentage. Until reaching this limit, the budget is shown in green.

budgetSafeZoneLimit

x

80

Notification dismiss time

It sets the time that the notification triggered by the Budget widget is visible for. The default value is in seconds.

notificationDismissTime

x

5

Number of budgets shown

It sets the maximum number of budgets shown.

maxBudgets

x

[a] Preferences that have this column checked are only editable for administrators.
