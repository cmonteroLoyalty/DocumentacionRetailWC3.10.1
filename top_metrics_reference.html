View top spending metrics

This page includes the following information:

  • Selector name

  • Required imports

  • Extension slots

  • Implementation example

  • Preferences

Selector name

This widget has the following selector name: <bb-top-metrics-widget>.

Required imports

The following imports are required to set up and customize this widget:

  • Widget module import

    Import the following modules into the module of your wrapper widget:

    import { TopMetricsWidgetModule } from '@backbase/retail-ang/pfm';

  • Widget component import

    Import the following components into the module of your wrapper widget:

    import { TopMetricsWidgetComponent } from '@backbase/retail-ang/pfm';

  • Data module imports

    Import the following data modules into the module of your wrapper widget:

    import { ProductSummaryDataHttpModule } from '@backbase/data-ang/product-summary';
import { TransactionsDataHttpModule } from '@backbase/data-ang/transactions';

bbTopMetricsCategoryItemCustomizable

This extension slot includes the following default template:

<ng-template bbTopMetricsCategoryItemCustomizable let-hostRef let-categoryTotalItem="context">
  <div class="category-total-item row d-flex" role="row" data-role="category-total-item">
    <div class="col-8 d-flex align-items-center">
      <!-- Icon -->
      <div *ngIf="categoryTotalItem.category" class="category-total-category">
        <span data-role="category-total-category" [ngClass]="hostRef.getCategoryIconClass(categoryTotalItem.category)"></span>
      </div>
      <!-- Category -->
      <div class="category-total-name ml-3">
        <span data-role="category-total-name">{{ categoryTotalItem?.category }}</span>
      </div>
    </div>

    <div class="col-4 d-flex flex-row-reverse align-items-center">
      <!-- Amount -->
      <div class="category-total-amount">
        <bb-amount-ui
          data-role="category-total-amount"
          [amount]="categoryTotalItem?.amount"
        ></bb-amount-ui>
      </div>
    </div>
  </div>
</ng-template>

Implementation example

The following examples show the implementation of this widget:

  • Wrapper widget module

    NewExtensionSlot.module.ts
    import { NgModule } from '@angular/core';
import { CommonModule } from '@angular/common';
import { TopMetricsWidgetModule } from '@backbase/retail-ang/pfm';
import { TopMetricsMySelectorComponent } from './NewExtensionSlot.component'

@NgModule({
  declarations: [TopMetricsMySelectorComponent],
  imports: [
    CommonModule,
    TopMetricsWidgetModule,
  ],
})
export class CustomAngModule {}

  • Wrapper widget component

    NewExtensionSlot.component.ts
    import { Component } from '@angular/core';
import { CopyRoutes } from '@backbase/core-ang';
import { TopMetricsWidgetComponent } from '@backbase/retail-ang/pfm';

@Component({
  selector: 'bb-my-selector',
  templateUrl: 'my-selector.component.html',
})
@CopyRoutes(TopMetricsWidgetComponent)
export class TopMetricsMySelectorComponent {
  // Include your customization here
}

    TopMetricsWidgetComponent is the name of the Top Metrics widget component.

  • Custom template

    NewExtensionSlot.component.html
    <bb-top-metrics-widget></bb-top-metrics-widget>

<!-- Override the empty state for the Top Metrics Widget -->
<ng-template bbExtensionSlot let-hostRef>
  <!-- Place your customization here -->
</ng-template>

The widget has the following preferences that managers or administrators can edit in Experience Manager:

Title and description Name Admin[a] Default value

Show period selector

showPeriodSelector

X

Metric type

Select either Transactions or Categories.

metricType

X

Transactions

List limit

Number of items in the list.

listLimit

X

[a] Preferences that have this column checked, are only editable for administrators.
