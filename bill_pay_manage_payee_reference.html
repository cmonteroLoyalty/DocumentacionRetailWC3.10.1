Selector name

This widget has the following selector name: <bb-billpay-manage-payee-widget>

Required imports

The following imports are required to set up and customize this widget:

  • Widget module import

    import { BillpayManagePayeeWidgetModule } from '@backbase/billpay-manage-payee-widget-ang';
Note
 For standalone use and testing of this widget, import BillpayEnrolmentWidgetModule in the app.module.ts of your app. All Bill Pay widgets are closely related to each other, so it is recommended to use them together as well.
