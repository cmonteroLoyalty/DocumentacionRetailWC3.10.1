Create online banking account

Preferences

The widget has the following preferences that managers or administrators can edit in Experience Manager:

Title and description Name Admin[a] Default value

Complete navigation

Sets a navigation path when the account creation is successfully completed.

output.complete

X

navigation:

Cancel navigation

Sets a navigation path when the user requests to cancel the enrollment process.

output.cancel

X

navigation:

Log in navigation

Sets a navigation path when the request is being made to navigate to the login screen.

output.login

X

navigation:

[a] Preferences that have this column checked, are only editable for administrators.

Access Control

The bank customer does not require any permissions to access this widget’s functionality.

Extension slots

This widget has the following extension slots:

  1. bbSelfEnrollmentCreateAccountCustomizable

  2. bbSelfEnrollmentCreateAccountFormFieldsCustomizable

  3. bbSelfEnrollmentCreateAccountSubmitErrorCustomizable

  4. bbSelfEnrollmentCreateAccountPasswordErrorCustomizable
