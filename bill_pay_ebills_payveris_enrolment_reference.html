Selector name

This widget has the following selector name: <bb-billpay-payveris-ebills-enrolment-widget>.

Required imports

The following imports are required to set up and customize this widget:

  • Widget module import

    Import the following modules into the module of your wrapper widget:

    import { BillpayPayverisEbillsEnrolmentWidgetModule } from '@backbase/billpay-payveris-ebills-enrolment-widget-ang';
    NOTE

    All Bill Pay widgets are closely related to each other, so it is recommended to use them together.
