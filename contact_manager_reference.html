Contact manager

Preferences

The widget has the following preferences that managers or administrators can edit in Experience Manager:

Title and description Name Editable for Default value

Max. number of navigation pages

The number of displayed page numbers including the currently visible one when using pagination as pagination type.

maxNavPages

Admin

3

Enable the creation of new contacts

Enables or turns off the creation of new contacts.

newContact

Admin

true

Number of contacts per page

The number of contacts loaded initially and when additional contacts are loaded.

pageSize

Admin

5
Untitled

Pagination type (pagination, load-more)

The number of contacts loaded initially and when additional contacts are loaded.

  • Load more: A Load more button is displayed under the list, which when clicked displays additional notifications. The button remains until there are no more entries to display

  • Pagination: An index of available pages is displayed under the list, which always provides links to the first and last page, as well as to a number of pages adjacent to the current one.

paginationType

Admin

pagination

Display avatars

Displays or hides avatars.

showAvatar

true

Account number format[depracated]

Sets the account number format to IBAN or BBAN.

accountType

Enable Creation of contact through Email

Enables or turns off contact creation through email.

createContactEmail

false

Enable Creation of contact through IBAN

Enables or turns off contact creation through IBAN.

createContactIBAN

true

Enable Creation of contact through Account Number

Enables or turns off contact creation through account number.

createContactAccount

false

Enable Creation of contact through Phone Number

Enables or turns off contact creation through phone number.

createContactPhone

false

bbContactsTopSectionCustomizable

You configure this extension slot using:

 <ng-template bbContactsTopSectionCustomizable let-hostRef let-item="context">
    <div class="col bb-stack">
      <div class="bb-stack__item bb-stack__item--fill">
        <bb-search-box-ui
          class="bb-toolbar__item"
          data-role="search-contact-textbox"
          placeholder="Search"
          i18n-placeholder="Search box placeholder | Placeholder for search box@@contact-manager.placeholder.search"
          [showSearch]="true"
          [showClear]="true"
          (submit)="hostRef.searchBoxItemSelected()"
          (clear)="hostRef.searchBoxItemSelected(true)"
          [formControl]="hostRef.searchTerm"
          (keyUp.enter)="hostRef.searchBoxItemSelected()"
        ></bb-search-box-ui>
      </div>
      <div class="bb-stack__item">
        <button
          bbButton
          i18n="New Contact Button | Button for creating a new contact@@contact-manager.button.newContact"
          data-role="new-contact-button"
          class="d-none d-lg-block bb-button-bar__button"
          [disabled]="!(hostRef.newContactAllowed | async)"
          (click)="hostRef.newContact(item.contacts.length)"
          type="button"
        >
          New Contact
        </button>
        <button
          bbButton
          i18n="New Contact Button | Button for creating a new contact@@contact-manager.button.newContact"
          data-role="new-contact-button"
          class="d-block d-lg-none bb-button-bar__button"
          [disabled]="!hostRef.newContactAllowed"
          (click)="hostRef.newContact(item.contacts.length)"
          type="button"
        >
          New Contact
        </button>
      </div>
    </div>
    <ng-template #rt let-result="result" let-term="term">
      <div class="text-truncate" [title]="result">
        <ngb-highlight [result]="result" [term]="term"></ngb-highlight>
      </div>
    </ng-template>
  </ng-template>

bbContactsListContainerCustomizable

You configure this extension slot using:

  <ng-template bbContactsListContainerCustomizable let-hostRef let-items="context">
    <div
      class="col"
      [ngClass]="{
        'd-block': hostRef.isListView | async,
        'd-lg-block d-none': !(hostRef.isListView | async)
      }"
    >
      <bb-contacts-list-widget
        [items]="items.contacts"
        [accountType]="hostRef.accountType$ | async"
        (selectedChange)="hostRef.select($event)"
        (formItemChange)="hostRef.edit($event)"
      >
      </bb-contacts-list-widget>
    </div>
  </ng-template>

bbContactsLoadMoreCustomizable

You configure this extension slot using:

<ng-template bbContactsLoadMoreCustomizable let-hostRef let-items="context">
  <bb-pagination-ui
    *ngIf="(hostRef.paginationType | async) !== 'load-more'"
    class="bb-pagination"
    [page]="(hostRef.pagination | async)?.from"
    [boundaryLinks]="false"
    [directionLinks]="true"
    [totalItems]="items?.count"
    [itemsPerPage]="hostRef.pageSize | async"
    [maxNavPages]="hostRef.maxNavPages | async"
    (pageChange)="hostRef.onPageChange($event)"
  ></bb-pagination-ui>
  <div class="p-5 text-center" *ngIf="(hostRef.paginationType | async) === 'load-more'">
    <button class="border text-dark btn btn-secondary" (click)="hostRef.loadMore()">Load more</button>
  </div>
</ng-template>
