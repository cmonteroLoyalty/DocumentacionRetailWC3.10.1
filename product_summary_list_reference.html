Selector name

This widget has the following selector name: bb-product-summary-list-widget

Required imports

The following imports are required to set up and customize this widget:

  • Widget module import

    Import the following modules into the module of your wrapper widget:

    import { ProductSummaryListWidgetModule } from '@backbase/product-summary-list-widget-ang';

bbProductSummaryCustomizable

This extension slot includes the following default template:

<ng-template
  bbProductSummaryCustomizable
  let-hostRef
  let-items="context">
  <ng-container
    bbProductSummaryHeaderCustomizable
    [bbHostRef]="hostRef"
    [bbTemplateContext]="items">
  </ng-container>

  <ng-container
    bbProductSummaryBodyCustomizable
    [bbHostRef]="hostRef"
    [bbTemplateContext]="items">
  </ng-container>
</ng-template>

This extension slot consumes the following object properties from its custom context:

Object property Description

context: ProductKinds

Contains list of available product summary items.

      Other (Default):
      This template can be used to add more custom productKinds.

      [bbHostRef]="hostRef"

    </ng-container>

  </ng-container>

             let-hostRef>

        </ng-container>
        <bb-icon-ui class="bb-product-kind__balance-icon "
                    name="toggle-{{!hostRef.isExpanded ? 'up' : 'down'}}"></bb-icon-ui>

    </div>

      </div>
    </div>
  </div>
</ng-template>
