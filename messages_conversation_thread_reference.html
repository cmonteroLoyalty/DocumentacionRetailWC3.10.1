Selector name

This widget has the following selector name: bb-messages-conversation-thread-widget-ang.

Required imports

The following imports are required to set up and customize this widget:

  • Angular Common module import:

    import { CommonModule } from '@angular/common';

  • Backbase Core module import:

    import { BackbaseCoreModule } from '@backbase/foundation-ang/core';

  • Widget module import:

    import { MessagesConversationThreadWidgetModule } from '@backbase/messages-conversation-thread-widget-ang';

Extension slots

This widget has the following extension slots:

  1. bbMessagesConversationThreadCustomizable

  2. bbMessagesUploadAttachmentsCustomizable

  3. bbMessagesManipulationConfirmModalCustomizable
