View amount left to spend

This page includes the following information:

  • Selector name

  • Required imports

  • Extension slots

  • Implementation example

  • Preferences

Selector name

This widget has the following selector name: <bb-left-to-spend-widget>.

Required imports

The following imports are required to set up and customize this widget:

  • Widget module import

    Import the following modules into the module of your wrapper widget:

    import { LeftToSpendWidgetModule } from '@backbase/retail-ang/pfm';

  • Widget component import

    Import the following components into the module of your wrapper widget:

    import { LeftToSpendWidgetComponent } from '@backbase/retail-ang/pfm';

  • Data module imports

    Import the following data modules into the module of your wrapper widget:

    import { ProductSummaryDataHttpModule } from '@backbase/data-ang/product-summary';
import { TransactionsDataHttpModule } from '@backbase/data-ang/transactions';

Extension slots

This widget has the following extension slots:

  1. bbLeftToSpendDataCustomizable

bbLeftToSpendDataCustomizable

This extension slot includes the following default template:

<ng-template bbLeftToSpendDataCustomizable let-hostRef let-spendingAnalysis="context">
  <div class="card text-white bg-success">
    <div class="card-body">
      <div class="text-center">
        <ng-container *ngIf="spendingAnalysis.leftToSpend">
          <p class="d-flex justify-content-center align-items-center mb-2">
            <span data-role="title-text" i18n="Header text|Text to display at the top of the left to spend amount@@left-to-spend.header">Left to spend</span>
            <bb-icon-ui class="ml-2" name="info" size="md" placement="bottom" [bbTooltip]="tooltip"></bb-icon-ui>
          </p>
          <bb-amount-ui
            data-role="left-to-spend-amount"
            class="h1 m-0 text-white"
            [amount]="spendingAnalysis.leftToSpend.amount"
            [currency]="spendingAnalysis.leftToSpend.currencyCode"
            [decimalPlaces]="0"
          >
          </bb-amount-ui>
        </ng-container>
        <p class="mt-2 mb-4">
          <span data-role="days-left-text" i18n="Next month|Message with the number of days until the next month@@left-to-spend.next.month"
          >{spendingAnalysis.daysLeft, plural, =0 {Tomorrow is next month} other {{{spendingAnalysis.daysLeft}} days until next month}}</span>
        </p>
      </div>
      <hr class="bg-white">
      <div *ngIf="spendingAnalysis.incomeToDate" class="d-flex justify-content-between align-items-center">
        <p data-role="income-to-date-text" class="my-1" i18n="This period's income|Label for this period's income value@@left-to-spend.period.label.income">Income</p>
        <bb-amount-ui
          data-role="income-to-date-amount"
          [amount]="spendingAnalysis.incomeToDate.amount"
          [showPlusSign]="true"
        >
        </bb-amount-ui>
      </div>
      <div *ngIf="spendingAnalysis.spentToDate" class="d-flex justify-content-between align-items-center">
        <p data-role="spent-to-date-text" class="my-1" i18n="This period's expenses|Label for this period's expense value@@left-to-spend.period.label.expense">Spent</p>
        <bb-amount-ui
          data-role="spent-to-date-amount"
          [amount]="spendingAnalysis.spentToDate.amount * -1"
        >
        </bb-amount-ui>
      </div>
      <div *ngIf="spendingAnalysis.schedulePayments && spendingAnalysis.schedulePayments.total" class="d-flex justify-content-between align-items-center">
        <p data-role="schedule-payments-text" class="my-1" i18n="This period's upcoming payments|Label for this period's upcoming payments amount@@left-to-spend.period.label.payments">Upcoming payments</p>
        <bb-amount-ui
          data-role="schedule-payments-amount"
          [amount]="spendingAnalysis.schedulePayments.total.amount * -1"
        >
        </bb-amount-ui>
      </div>
    </div>
  </div>
</ng-template>

Implementation example

The following examples show the implementation of this widget:

  • Wrapper widget module

    NewExtensionSlot.module.ts
    import { NgModule } from '@angular/core';
import { CommonModule } from '@angular/common';

import { LeftToSpendMySelectorComponent } from './NewExtensionSlot.component'

@NgModule({
  declarations: [LeftToSpendMySelectorComponent],
  imports: [
    CommonModule,
    LeftToSpendWidgetModule,
  ],
})
export class CustomAngModule {}

    NewExtensionSlot.component.ts
    import { Component } from '@angular/core';
