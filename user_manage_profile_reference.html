Access Control

The user does not require any permissions to access this widget’s functionality.

Extension slots

This widget requires you to add providers when you customize its extension slots. To enable your extension slots, add the following providers to your custom component:

  • ManageProfilePreferencesService from @backbase/user-common

  • UserManageProfileService from @backbase/user-manage-profile-widget

  • ManageProfileService from @backbase/user-common

This widget has the following extension slots:
