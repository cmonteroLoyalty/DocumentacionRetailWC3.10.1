Guard bill pay enrollment

  • Selector name

  • Required imports

  • Extension slots

  • API

  • Implementation example

  • Preferences

Selector name

This widget has the following selector name: <bb-billpay-guard-container>.

Required imports

The following imports are required to set up and customize this widget:

  • Widget module import

    import { BillpayGuardContainerModule } from '@backbase/billpay-guard-container-ang';
    NOTE

    All Bill Pay widgets are closely related to each other, so it is recommended to use them together.

Extension slots

bbBillpayGuardContainerCustomizable

Configure this extension slot using:

<ng-template bbBillpayGuardContainerCustomizable let-host>
  <bb-billpay-guard (goToEnrolment)="hostRef.guardResult($event)"></bb-billpay-guard>
  <bb-area [children]="children | async"></bb-area>
</ng-template>

API

The following table shows the public class members of the host component that you access from each extension slot:

Component Extension slot identifier Public class member Description

BillpayGuardContainerComponent

bbBillpayGuardContainerCustomizable

readonly hostRef: BillpayGuardContainerComponent

Component context.

children: Observable<Array<ItemModelTree>>;

The child elements in the page model.

goToEnrolment: EventEmitter<string>;

The @Output() to emit the route.

guardResult(route: string): void;

The function to emit the route.

Implementation example

The following examples show the implementation of this widget:

  • Wrapper widget module

    import { NgModule } from '@angular/core';
import { CommonModule } from '@angular/common';
import { BackbaseCoreModule } from '@backbase/foundation-ang/core';
import { MyCustomWidgetComponent } from './my-custom-widget.component';
import { BillpayGuardContainerModule } from '@backbase/billpay-guard-container-ang';
import { BillpayDataHttpModule } from '@backbase/data-ang/billpay';
import { BillpayCommonModule } from '@backbase/billpay-common';

@NgModule({
  declarations: [MyCustomWidgetComponent],
  imports: [
    CommonModule,
    BackbaseCoreModule.withConfig({
      classMap: { MyCustomWidgetComponent },
    }),
    BillpayDataHttpModule,
    BillpayGuardContainerModule,
    BillpayCommonModule,
  ],
})
export class MyCustomWidgetModule {}
Screen reader support enabled.

  • Wrapper widget component

    @Component({
  selector: 'my-custom-widget',
  template: '<bb-billpay-guard-container></bb-billpay-guard-container>',
  styles: [],
})
export class MyCustomWidgetComponent implements OnInit {
  constructor() {}

  ngOnInit() {}
}
